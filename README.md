<p>A word guessing game built using python with a command-line interface.</p>

<p>In order to load the game, make sure all code files exist within the same directory.</p> 
  
<h2>This includes:</h2>
<p>main.py</p>
<p>wrdguessr.py</p>
<p>words.txt (CSV with words separated by hint)</p>
<p>scoreboard.txt (Can be a empty file)</p>

<br />

<p>Execute the main.py file to launch the game.</p>
